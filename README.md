# Sales Prediction Dashboard

## Project Description

The Sales Prediction Dashboard is an interactive web application built using Streamlit. It analyzes historical sales data, provides meaningful visualizations, and predicts future sales trends using machine learning. The dashboard helps users explore sales performance and supports data-driven business decisions.

## Features

* Interactive and user-friendly Streamlit dashboard
* Sales data visualization using charts and graphs
* Sales prediction using a Machine Learning model
* Key Performance Indicators (KPIs) for business insights
* Data filtering and analysis
* Download sales report functionality
* Responsive and easy-to-use interface

## Tools & Technologies

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Plotly
* Joblib
* Git & GitHub
